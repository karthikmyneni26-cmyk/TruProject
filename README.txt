TRU PROJECTS FLASK WEBSITE

1. Open terminal in this folder.
2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python app.py

4. Open:
   http://127.0.0.1:5000

5. Admin dashboard:
   http://127.0.0.1:5000/admin

The SQLite database tru_projects.db is automatically created when the app starts.
